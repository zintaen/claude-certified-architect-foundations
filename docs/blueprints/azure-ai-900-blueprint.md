## Learning objectives (outline, own words)

### AI workloads and considerations

- Identify common AI workload types and responsible-AI considerations

### Machine learning on Azure

- Describe supervised vs unsupervised patterns and Azure ML concepts at a fundamentals level

### Computer vision workloads on Azure

- Recognize vision service scenarios (classification, detection, OCR) without vendor exam prose

### Natural language processing on Azure

- Map language understanding / translation / summarization scenarios to Azure NLP capabilities

### Generative AI workloads on Azure

- Describe generative AI app patterns and grounding / safety concerns at a fundamentals level

## Out of scope

- Recalled live exam items
- Non-AI Azure certs (AZ-900 etc. — later wave)
