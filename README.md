# Claude Certified Architect — Foundations (CCAF) Mock Exam

A single-page, offline-friendly mock exam for the **Claude Certified Architect — Foundations** credential, built from what I learned preparing for and passing the real exam.

- **60 questions**, based on the real exam blueprint
- **120-minute timer** (with an untimed practice mode)
- **Scenario-grouped**: related questions stay together; the overall order is shuffled each run
- **Review mode** unlocks the explanations after you finish (and, in timed mode, beat the clock)
- **Mobile-responsive**: phone, tablet, laptop
- **Self-contained**: one `index.html` file, no build step, no backend

## Try it

Open `index.html` in any modern browser, or drop it on a static host (GitHub Pages, Netlify, Vercel).

## Why I built it

I sat the real CCAF exam, passed it, and wrote up my playbook on Reddit. A lot of people asked for a realistic set of practice questions to pressure-test their understanding before going in. This is that set.

## What's in here

- `index.html` — the full mock exam (HTML + CSS + JS + base64-encoded question bank)
- `questions.txt` — the plain-text question source (what the runtime decodes at start-up)
